## How to set it up locally
**Clone the repository**
```
git clone <repo_url>
```

**Create and activate a virtual environment**
```
conda create --name <name> python=3.7
conda activate <name>
```

**Download the requirements**
```
pip install requirements.txt
```

**Run the app**
 ```
 streamlit run <filename>
 ```
